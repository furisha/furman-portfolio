Testing

Functional testing

Perform tests to confirm that an application works according to the requirements.

Smoke testing

Perform tests for basic functionality once the building has been deployed.

UI testing

Perform tests for user interface functionality to check how fully and correctly it meets the requirements.

Regression testing

Perform tests to make sure that any new changes are working as expected and do not affect the previous or existing modules and functionality.

User Acceptance testing

Perform the final assessment before the product is ready for release.

Exploratory testing

Extending the test coverage by going beyond the predefined test suite.

System testing

Non-functional testing

Security testing

Uncover vulnerabilities, risks, and threats of malicious attacks from internal and external sources by simulating malicious attacks on a given system.
